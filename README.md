## Integrantes

- Paco | Desarrollador Técnico

## Descripción
Este proyecto analiza registros de ventas diarias de una pequeña empresa
para identificar tendencias y generar indicadores clave de desempeño comercial.

## Dataset
Archivo CSV con registros diarios de ventas que incluye:
- Producto vendido
- Cantidad vendida
- Precio unitario
- Fecha de venta

## Qué analiza el script
- Ventas totales del período
- Producto más vendido
- Evolución de ventas por mes

## Cómo ejecutar
1. Clonar el repositorio
2. Abrir `scripts/analisis_ventas.py` en Google Colab
3. Los resultados y gráficos se guardan en `/resultados`

## Requisitos
- Python 3
- pandas
- matplotlib
